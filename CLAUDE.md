## This vault
This is my **personal** brain. It is a different vault from my work brain and the two
must never be mixed. If you are ever asked to pull in work material, stop and ask.

Folders:
- `Inbox/`    — raw captures land here, often from my phone. Process these first.
- `Notes/`    — durable, linked knowledge.
- `Projects/` — one folder per active personal project.
- `Archive/`  — finished or dormant.

## How to work here
- When something is sitting in `Inbox/`, read it, decide where it belongs, move it,
  and add `[[wikilinks]]` to related notes.
- Prefer linking over duplicating. If a concept already has a note, link to it
  instead of writing a second one.
- When you create or substantially change a note, tell me what changed so I can review.
- This vault is version-controlled with git, so you may edit freely — I can roll back.
